# Neon Aura AR

A real-time **Augmented Reality (AR) hand tracking web application** that delivers an interactive, immersive visual experience using gesture recognition, dynamic particle systems, and audio feedback.

---

## 📌 Overview

Neon Aura AR leverages **computer vision** and **browser-based rendering** to track hand movements and translate them into responsive visual and audio effects. The application demonstrates the integration of real-time input processing with creative coding techniques.

---

## 🚀 Key Features

* Real-time hand tracking (up to 2 hands) using MediaPipe
* Gesture recognition (pinch, open hand, fist detection)
* Dynamic particle system with shockwave effects
* Multiple visual themes (Rainbow, Cyberpunk, Lava, Ocean, Galaxy)
* Interactive audio feedback using Web Audio API
* Performance monitoring (FPS, gesture state, hand count)
* Responsive full-screen canvas rendering

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Libraries:** MediaPipe Hands
* **APIs:** Canvas API, Web Audio API
* **Tools:** Git, GitHub

---

## ⚙️ Installation & Usage

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Grant camera access when prompted
4. Click **Enter Experience** to start the application
